# 腾讯云计费方式选择指南

## 📊 计费方式对比

### 按量计费（后付费）

**特点：**
- 按小时计费，每小时扣费
- 随时创建，随时销毁
- 灵活调整配置

**适用场景：**
- 测试环境
- 开发环境
- 短期项目
- 流量不确定的新项目

### 包年包月（预付费）

**特点：**
- 预付费，一次性支付
- 长期使用有折扣
- 配置相对固定

**适用场景：**
- 生产环境
- 长期稳定运行
- 流量可预测
- 成本可控

## 🎓 决策流程

### 问题 1: 项目是否已经稳定运行？

**是** → 继续问题 2
**否** → 使用**按量计费**（3 个月后重新评估）

### 问题 2: 预计运行时间？

**< 3 个月** → 使用**按量计费**
**3-12 个月** → 使用**包月**
**> 12 个月** → 使用**包年**（最优）

### 问题 3: 配置是否可能调整？

**不太可能** → 使用**包年**（最优）
**可能调整** → 使用**包月**
**经常调整** → 使用**按量计费**

### 问题 4: 预算是否充足？

**充足** → 使用**包年**（最优）
**紧张** → 使用**按量计费**（分期付款）

## 🔄 计费方式转换

### 按量转包年包月

**步骤：**
1. 登录腾讯云控制台
2. 进入对应服务（Redis/MySQL）
3. 选择实例 → 更多 → 转包年包月
4. 选择购买时长
5. 支付费用

**注意：**
- ✅ 可以随时转换
- ✅ 按量计费的使用时长会计入
- ✅ 转换后立即生效
- ⚠️ 转换后不能再改回按量计费

### 包年包月续费

**自动续费（推荐）：**
1. 实例详情 → 续费管理
2. 开启自动续费
3. 设置续费时长（1 年）

**手动续费：**
1. 到期前 7 天会收到通知
2. 进入续费管理
3. 选择续费时长
4. 支付费用

**注意：**
- ⚠️ 到期后会保留 7 天
- ⚠️ 7 天后会销毁实例
- ⚠️ 数据无法恢复

**获取途径：**
- 新用户注册赠送
- 活动期间领取
- 企业认证赠送

**使用方法：**
- 购买时自动抵扣
- 可叠加使用
- 注意有效期

**常见活动：**
- 双11、618 大促
- 新用户专享
- 企业采购优惠

**折扣力度：**
- 新用户首购：3-5 折
